### Combos
```
s + l = /
shift + s + l = \
d + K = tab
f + j = |

x + c = Ctrl + C (Copy)
c + v = Ctrl + V (Paste)
x + v = Ctrl + X (Cut)
```
### Magic Key
```
y + x + c + v = switch between Mac and Pc
```
